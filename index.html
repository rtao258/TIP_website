Hangman

The hangman project focused on basic elements of Python such as if/else statements and for loops. The goal was to build a functioning hangman game. I wrote a Python module that can be used to make your own hangman game easily.

The following Python code is part of the hangman module.

@staticmethod
def guess_char(parsed_chars, guess):
    """ Modifies the parsed char list as necessary for a guessed character in the word """
    correct = False
    solved = True
    # First, validate char
    if not Text.validate_char(guess):
        print("That's not a single letter! Try again.")
        correct = True
        return parsed_chars, correct
    # Look for matching characters
    for i in range(len(parsed_chars)):
        char = parsed_chars[i][0]
        if char == guess:
            correct = True
            parsed_chars[i][1] = True
    # Check if all characters are guessed correctly
    for char in parsed_chars:
        if char[1] == False:
            solved = False
    if solved:
        print("You solved the hangman puzzle!")
    # Print a message to the console
    elif correct:
        print("\"{}\" is correct!".format(guess))
    else:
        print("\"{}\" is incorrect!".format(guess))
    return parsed_chars, correct, solved

def init_board(screen, render, space, center=True):

# First, define some local variables
# Initialize the board which will hold the boxes
board = [[],
         [],
         []]

# Unpack the screen size tuple
screen_x = screen.get_size()[0]
screen_y = screen.get_size()[1]
# Unpack render to get values needed to render identical boxes
size = render[0]
border = render[1]
color = render[2]
# Unpack the space tuple for spacing between boxes
margin = space[0]
padding = space[1]
# Get offsets for proper centering if necessary
if center:
    table_width = 3 * size
    x_offset = (screen_x - table_width)/2
    y_offset = (screen_y - table_width)/2 + padding
else:
    x_offset = 0
    y_offset = padding

# Create the boxes and put them "on" the board
for row in range(3):
    for column in range(3):
        x = row * (size + 10) + x_offset
        y = column * (size + 10) + y_offset
        board[row].append(Box(screen, (x, y), (size, border, color), (False, None)))
print(board)
return board

Pac-Man

The Pac-Man project focused on topics in computer science, particularly search algorithms. The goal was to write an AI for Pac-Man to eat the dots. This used two kinds of searching algorithms, Depth-First Search and Breadth-First Search.

The following algorithm implemented in Python was used in both Depth-First Search and Breadth-First Search.

def genericSearch(problem, structure):
    fringe = structure();
    expanded = set();
    fringe.push((problem.getStartState(), [], 0));

    while not fringe.isEmpty():
        curState, curMoves, curCost = fringe.pop();

        if (curState in expanded):
            continue;

        expanded.add(curState);

        if problem.isGoalState(curState):
            return curMoves;

        for state, direction, cost in problem.getSuccessors(curState):
            fringe.push((state, curMoves + [direction], curCost));
    return [];
